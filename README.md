# analise-vendas

Análise de Dados de Vendas
Descrição

Este projeto tem como objetivo realizar uma análise exploratória de dados de vendas, buscando identificar padrões, tendências e métricas relevantes para apoio à tomada de decisão. Serão aplicadas técnicas de limpeza, transformação e análise de dados utilizando Python e SQL.

Tecnologias Utilizadas
Python (Pandas, Matplotlib, Seaborn)
SQL
Jupyter Notebook
Objetivos do Projeto
Realizar o tratamento e a limpeza dos dados
Analisar indicadores como faturamento, ticket médio e volume de vendas
Identificar padrões de comportamento de clientes
Desenvolver visualizações para facilitar a interpretação dos dados
Estrutura do Projeto
data/: conjunto de dados utilizado no projeto
notebooks/: análises e explorações em Jupyter Notebook
images/: gráficos e visualizações geradas
